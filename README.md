# Projeto de Laboratório - Sistema para uma Rede de Cinemas
<br>
<h2> Descrição</h2>
Uma rede de cinemas deseja informatizar o processo de reserva de ingressos em suas diversas unidades. O objetivo é criar um sistema que permita que os clientes consultem os filmes em cartaz, escolham sessões e assentos disponíveis, realizem reservas e recebam seus ingressos de forma prática e segura. <br><br>
O sistema deve também permitir que administradores gerenciem filmes, sessões e salas em diferentes locais, garantindo que os assentos não sejam reservados em duplicidade e que o controle de disponibilidade seja confiável.<br><br>
Além disso, a rede de cinemas possui um programa de fidelidade com benefícios especiais:<br>
• Clientes podem reservar os melhores assentos se realizarem a compra com até dois dias de antecedência.<br>
• Cada cliente cadastrado no programa recebe um ingresso gratuito no mês de seu aniversário.<br><br>
O sistema deve ainda contemplar a comercialização de ingressos de meia-entrada, conforme legislação vigente, para:<br>
• Estudantes devidamente identificados.<br>
• Pessoas maiores de 65 anos.<br>
• Professores da rede pública e privada.<br>
<br><br>
<h2>Escopo </h2>
• Clientes poderão consultar filmes e sessões disponíveis em diferentes salas e locais.<br>
• Clientes poderão reservar assentos e obter ingressos digitais ou impressos.<br>
• Administradores poderão cadastrar e gerenciar filmes, sessões e salas em toda a rede.<br>
• O sistema deve controlar a disponibilidade de assentos em tempo real.<br>
• O programa de fidelidade deve aplicar regras especiais de reserva e benefícios.<br>
• O sistema deve gerar relatórios de vendas e renda por sessão.<br>
• O sistema deve enviar relatórios obrigatórios para a ANCINE (Agência Nacional do Cinema).<br>
• O sistema deve registrar, para cada sessão, a quantidade de ingressos vendidos e a renda obtida.<br>
• O sistema deve permitir a emissão de relatórios consolidados por filme, sala, cinema e período.<br>
• O sistema deve gerar relatórios automáticos após o término de cada sessão.<br>
• O sistema deve manter histórico de reservas e vendas para auditoria.<br>
• O sistema deve registrar a quantidade de ingressos de meia-entrada vendidos por categoria (estudantes, maiores de 65 anos, professores).<br>
• O sistema deve aplicar corretamente regras de meia-entrada.<br>
• O sistema deve integrar-se com um serviço da ANCINE para obter a lista oficial de filmes disponíveis.<br>
• O sistema deve registrar quais títulos foram disponibilizados em cada sala, com base na lista oficial da ANCINE.<br>
• O administrador poderá selecionar títulos dessa lista e disponibilizá-los em suas salas.<br>
